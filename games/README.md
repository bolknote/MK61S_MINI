# MK61 game files

Copy these `.m61` files to the MK61S USB disk.

`Bumblebee Fly Trap.m61` is the trap-driven variant of `Bumblebee Fly.m61`.
It intercepts the two `RECE55` pauses at program addresses 31 and 44, prints
the calculator state to the terminal, and executes the former manual `В/О`,
`С/П` continuation automatically. The original file is kept unchanged.

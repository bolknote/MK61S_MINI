# MKC — файловый менеджер MK61s

Версия документа: 21.07.2026

`tools/mkc.sh` — двухпанельный терминальный файловый менеджер в стиле Norton
Commander. Левая панель показывает каталог компьютера, правая — файловую
систему C5 калькулятора через его USB-терминал.

```bash
tools/mkc.sh
tools/mkc.sh --port /dev/cu.usbmodem1234
tools/mkc.sh --local ./programs
```

Нужен Bash 3.2 или новее и `arduino-cli`. Отдельные `dialog`, `whiptail` и
редактор не используются. `arduino-cli` находит CDC-устройство STM32 с
VID:PID `0483:5740` и держит один сеанс monitor на всё время работы. Если
автоопределение не удалось, MKC спрашивает имя последовательного порта.

Последний локальный каталог и порт сохраняются в git-игнорируемом файле
`.mkc.conf`. Правая панель всегда начинает с корня C5.

## Интерфейс

Окно центрируется, ограничено 92×25 символами и не закрашивает остальной
терминал. Палитра фиксирована в классических DOS-цветах Norton Commander:
тёмно-синий `#000080`, cyan и белый. Список каждой панели многоколоночный;
смена выбранной строки перерисовывает только старую и новую ячейки.

| Клавиша | Действие |
| --- | --- |
| `Tab` | Сменить активную панель. |
| Стрелки, `Home`, `End`, `PgUp`, `PgDn` | Навигация. |
| `Enter`, `Backspace` | Войти в каталог или подняться выше. |
| `Space`, `Insert` | Отметить несколько объектов. |
| `F1` | Справка. |
| `F3` | Встроенный просмотр текста или hex без редактирования. |
| `F5` | Копирование между компьютером и калькулятором. |
| `F6` | Переименование или перемещение в активной панели. |
| `F7` | Создание каталога. |
| `F8` | Удаление после подтверждения. |
| `F9` | Ёмкость C5 и сведения об устройстве. |
| `F10` | Выход. |
| `Ctrl-R` | Обновить обе панели. |

Отметки применяются к F5 и F8. Каталог копируется рекурсивно. Перед загрузкой
дерева MKC сначала проверяет все имена, форматы и размеры, поэтому ошибка не
оставляет на устройстве молча урезанную копию.

## Поддерживаемые файлы

На калькулятор копируются `.m61`, `.foc`, `.tbi`, `.txt`, `.state.txt`, `.fmk`
и `.wbmp`. Принимаются старые расширения `.t1`, `.m2` и `.wbm`; после записи
они отображаются с каноническим расширением. Обычные файлы ограничены 1536
байтами, WBMP — 1600 байтами, basename — 31 байтом UTF-8.

Символические ссылки, неизвестные расширения, слишком большие файлы и
недопустимые имена показаны серым со знаком `×`. F5 для них сразу блокируется
и нижняя строка объясняет причину. F3, F6 и F8 остаются доступны: например,
неверное расширение можно исправить переименованием.

## Проверка без устройства

Для разработки и автоматических тестов правая панель может быть сопоставлена
с обычным каталогом:

```bash
tools/mkc.sh --mock /tmp/mk61-device --local ./programs
```

Режим `--mock` не открывает последовательный порт. Реальная передача использует
машинные команды `fsget`/`fsput`, описанные в `MK61s-mini-Terminal.md`, с
96-байтовыми HEX-блоками и контрольной суммой POSIX `cksum`. Этот протокол не
зависит от shell и рассчитан на последующий PowerShell-порт для Windows.
